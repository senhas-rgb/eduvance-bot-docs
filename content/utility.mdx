# 10. Utility Commands

General utility commands for help and bot information.

---

## 10.1 `/help`

Show all commands or detailed help for a specific command.

```discord filename="Discord" copy
/help [command]
```

> [!NOTE]
>
> This command is available to all members.

#### Behavior

- **Without argument:** lists commands by category (moderation, warnings, punishments, history & statistics, config & utilities, reputation, utilities)
- **With `command`:** shows syntax, purpose, and key notes

> [!TIP]
>
> Use `/help <command>` to get detailed information about any specific command.

---

## 10.2 `/about`

Overview of the bot, features, and performance metrics.

```discord filename="Discord" copy
/about
```

> [!NOTE]
>
> This command is available to all members.

#### Behavior

- **Bot information:** name, version, overall description, key feature list
- **Performance metrics:** bot latency, API latency
- **Visual elements:** avatar, banner image
- **Feature categories:** Educational resources, advanced moderation, server insights, community management

> [!TIP]
>
> Share this command with new members to help them understand what Eduvance Bot can do for your server.
